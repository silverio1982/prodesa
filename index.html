RECORDAR CLAVE DE INGRESO A LA APLICACIÓN

Si no recuerda la clave de ingreso a la aplicación por favor ingrese el nombre de usuario con el cual se registra usualmente. El sistema le enviará un código aleatorio al correo electrónico registrado con el usuario para la verificación del cambio de contraseña.

Recuerde que si hizo más de intentos fallidos de registro en la aplicación, el sistema automáticamente bloquea su cuenta. Para volver a tener acceso a ella será necesario que se ponga en contacto con el administrador del sistema, para que active el usuario y le asigne nuevamente su clave.

Es también importante que reporte cualquier anomalía en este proceso. Como por ejemplo que usted no haya intentado ingresar a la aplicación y su usuario ya esté bloqueado.
